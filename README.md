Proyecto de [Next.js](https://nextjs.org/) Se consume una api externa (reqres.in)

## Empezando

Arrancar servidor de desarrollo:

```bash
npm run dev

```

Abrir [http://localhost:3000](http://localhost:3000) en el navegador para ver el resultado.
